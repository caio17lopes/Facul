    Criptografia em codificar os dados armazenados, de modo que apenas as pessoas autorizadas
    possam acessa-los
    É fundamental para garantir a segurança, protegendo a confidencialidade e a integridade das informações

    Criptografia    
        Utiliza dados matematicos para transforma os dados em texto especifico 
        Dados em repouso e em transito
        Possibilita a proteção ddos dados durante todo ciclo de vida
        Armazenamento, transmmissão, processamento 
        Maioria dos SGBDs possui soluções nativas 
        Implementação correta  depende do ambiente especifico de BD e dos requisitos de segurança 
        Pode afetar a performançe do sistema 

    Armazenamento de senhas 
        Deve ser realizado de forma criptografada 
        Uso de funções hash
        Valor irreversivel e unico 
        Uso de senhas complexas 
        Alteradas regularmente 

    Tipos de dados 
        Garantem a integridade dos dados
        Evitam erros de incersão e perda de dados 
        Previnem ataques de injeção e codigo malicioso
